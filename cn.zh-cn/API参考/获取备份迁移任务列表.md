# 获取备份迁移任务列表<a name="drs_03_0004"></a>

## 功能介绍<a name="section24649069"></a>

获取当前备份迁移任务列表，不包含已删除的任务。

## URI<a name="section20515029"></a>

-   URI格式

    PATH：/\{version\_id\}/\{project\_id\}/replication\_tasks

    Method：GET
